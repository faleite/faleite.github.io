
# Bem-vindo ao meu blog!

Este é um exemplo de blog feito com **HTML**, **CSS** e **JavaScript**, onde os posts são escritos em Markdown!

## Como funciona?

O blog usa a biblioteca [Marked.js](https://marked.js.org/) para converter Markdown em HTML automaticamente.

### Recursos do Markdown

Você pode usar:

- **Negrito** e *itálico*
- Listas numeradas e com marcadores
- Links e imagens
- Blocos de código
- E muito mais!

## Exemplo de código

```javascript
function saudacao(nome) {
    return `Olá, ${nome}!`;
}

console.log(saudacao("Mundo"));
```
